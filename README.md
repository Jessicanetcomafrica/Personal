## Love, Compiled

This workspace now includes a small static webpage you can send as a romantic link.

### Run it locally

Open `index.html` in a browser.

### Personalize the link

You can add query parameters to the URL so the page addresses the person directly:

`?to=Their%20Name&from=Your%20Name`

Example:

`index.html?to=Alex&from=Jordan`

### Share it online

To turn it into a real link, the easiest options are:

1. Upload these files to GitHub and enable GitHub Pages.
2. Drag the folder into Netlify Drop.
3. Upload it to Vercel as a static site.
